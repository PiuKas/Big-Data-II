## 1-Recursivas
Es una funcion que se usa muchas veces
```Python
def loquesea(): #definimos la funcioj
	print(hola)
loquesea() #llamamos la función
```
Podemos pasar argumentos varios
```Python
variable="hola"
def loquesea(variable):
	print(variable)
loquesea(variable)
#imprimirá hola
```
